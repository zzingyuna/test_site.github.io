# 앱  


* 딥링크  
1. 다이렉트 딥링크  
  - 앱이 있는 경우 : 앱 내 특정 페이지로 이동  
  - 앱이 없는 경우 : 각각의 앱 스토어로 이동  

2. 디퍼드 딥링크  
  - 앱이 있는 경우 : 앱 내 특정 페이지로 이동  
  - 앱이 없는 경우 : 앱 스토어로 이동 후 유저가 설치 및 실행하면 앱 내 특정 페이지로 이동  

3. 다이나믹 링크(원링크)
  - 앱이 있는 경우 : 앱 내 특정 페이지로 이동  
  - 앱이 없는 경우 : 앱 스토어로 이동 후 유저가 설치 및 실행하면 앱 내 특정 페이지로 이동  
  디퍼드 딥링크는 os별로 딥링크를 구현해야하지만 다이나믹 링크는 단 하나의 url로 각각의 os에 맞게 자동으로 앱 내 특정페이지로 이동할 수 있다
